# House-Price-Prediction
# 🏠 House Price Prediction

## 📌 Project Overview

This project aims to predict house prices using **Machine Learning (Linear Regression)** based on various features such as area, number of rooms, location, and other relevant factors.

The goal is to build a predictive model that can estimate house prices accurately.

---

## 🚀 Features

* Data preprocessing and cleaning
* Exploratory Data Analysis (EDA)
* Feature selection and transformation
* Model training using Linear Regression
* Model evaluation using performance metrics
* Visualization of results

---

## 🛠️ Technologies Used

* Python 🐍
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

---

## 📂 Dataset

The dataset contains information about house properties such as:

* Area
* Number of bedrooms
* Bathrooms
* Location
* Price (Target Variable)

---

## 📊 Model Evaluation Metrics

The model performance is evaluated using:

* **MAE (Mean Absolute Error)**
* **MSE (Mean Squared Error)**
* **RMSE (Root Mean Squared Error)**
* **R² Score (Coefficient of Determination)**

---

## 📈 Results

The model provides reasonably accurate predictions of house prices based on input features.
